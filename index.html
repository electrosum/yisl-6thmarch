Project Title: Stellar explorer

Tagline: Discover the secrets of stars

Description

Stellar Explorer is an immersive space exploration experience that allows users to travel through the universe virtually and learn about celestial bodies in an interactive way. Using technologies like AR/VR, users can explore planets, stars, galaxies, and space missions as if they are actually in space. The project provides detailed information, 3D visualizations, and simulations of cosmic phenomena, helping users understand astronomy and space science in a more engaging and educational manner. It aims to make learning about the universe exciting, interactive, and accessible to everyone.

Project Title: Traffic Police Training Using Virtual Reality (VR)

Tagline: Sharper Skills. Safer Streets.

Description

The Traffic Police Training using VR project is a virtual simulation that helps traffic officers practice managing road intersections in a safe digital environment. It simulates realistic traffic conditions with vehicles and pedestrians moving through a city junction. Users can control traffic flow using interactive controls such as allowing directional traffic, stopping vehicles, and enabling pedestrian crossings. This system provides a safe and effective way to train officers in traffic management and decision-making.

Team Name: Cipher (Team No. 5)
Team Members: Krishna Kanhaiya, KV Anagha, Gagana Gopinath, Maryam Mohamed Yahya, Athidhi V, Athena P Dinesh, Anusree P
